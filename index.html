Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes.

  • ValidadeApp: aplicativo web para controle de validades.

    Projeto em andamento desenvolvido em HTML, CSS e JavaScript para cadastrar produtos, acompanhar datas de validade e classificar itens vencidos, críticos, em atenção ou seguros. O app conta com autenticação, níveis de acesso, histórico, suporte PWA e já está sendo utilizado no meu trabalho atual, enquanto sigo trabalhando em melhorias de sincronização para múltiplos dispositivos.

    Ver projeto →

  • Sistema de Banco com nome fictício Nub.

    Em meu curso de UX e UI Design, desenvolvi um projeto de sistema bancário fictício chamado Nub, onde criei a interface e a experiência do usuário utilizando o Figma, aplicando princípios de design centrado no usuário para garantir uma navegação intuitiva e agradável.

    Ver no Figma →

  • Desenvolvimento de uma lagoa de peixes digitais animados.

    Consiste em um projeto de desenvolvimento de uma lagoa de peixes digitais animados, onde utilizei tecnologias de animação pré calculada e HTML para criar um ambiente virtual, permitindo aos usuários observar os peixes em tempo real, proporcionando uma experiência visualmente envolvente e relaxante.

  • O planeta Terra em um ângulo pouco visto por nós.

    Aplicando cálculos de física pré-programados, criei um projeto de visualização do planeta Terra junto a sua Lua, utilizando tecnologias de animação e HTML para criar uma representação realista do planeta e permitir interação com o ângulo da câmera, permitindo aos usuários explorar a Terra de uma perspectiva única e fascinante.
